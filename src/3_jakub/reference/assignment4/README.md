# Assignment 4 Reference Files

This directory contains the original reference material that was provided for
Assignment 4. These files are intentionally kept outside the live `include/`
tree so they do not look like active kernel headers or sources.

Purpose:
- preserve the original starter material for traceability
- keep the active kernel architecture separate from attached templates
- make it easier to compare the provided examples with the adapted
  implementation used in `src/3_jakub/src` and `src/3_jakub/include`

The active kernel build does not compile files from this directory.
